.mfp

Modals created via Magnific Popup library.



Basic modal

Create any html-content at the end of the page and wrap it into:
<div class="window mfp-hide" id="foo-bar-baz">...<div> at the end of the page and id.

To fire this popup create a link with mfp-handler class targeting this modal:
<a class="mfp-handler" href="#foo-bar-baz">Click me</a>
